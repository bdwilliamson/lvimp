# Format a `lvim` object

Format a `lvim` object

## Usage

``` r
# S3 method for class 'lvim'
format(x, digits = 3, ...)
```

## Arguments

- x:

  the `lvim` object of interest

- digits:

  the number of digits to format to

- ...:

  other options, see the generic `format` function

## Value

A formatted `lvim` object for printing.
